# CrownLedger Bank

A professional online banking website inspired by Chase, Wells Fargo, and Bank of America. Built with pure HTML, CSS, and JavaScript — no frameworks required.

---

## Overview

CrownLedger Bank is a fully functional, multi-page static banking website featuring a realistic account dashboard, transaction history, fund transfers, bill pay, and account management. Designed with a clean, trustworthy financial aesthetic using navy blues, whites, and subtle gold accents.

## Deployment

### GitHub Pages
1. Create a new repository on GitHub
2. Upload all files to the root directory
3. Go to **Settings → Pages**
4. Select **Deploy from a branch** → `main` / `root`
5. Your site will be live at `https://yourusername.github.io/crownledger-bank/`

### Custom Domain (Optional)
1. Add a `CNAME` file with your domain
2. Configure DNS A records to GitHub Pages IPs:
   - `185.199.108.153`
   - `185.199.109.153`
   - `185.199.110.153`
   - `185.199.111.153`
3. Add CNAME `www` → `yourusername.github.io`
